# documents

Documents for nbtca.

## How to use

### Install dependencies

1. Install Node.js
2. Install pnpm
   ```bash
   npm install -g pnpm
   ```
3. Install dependencies
   ```bash
    pnpm install
   ```

### Start development server

```bash
pnpm docs:dev
```

### Build for production

```bash
pnpm docs:build
```

### Lint code

```bash
pnpm lint
```
